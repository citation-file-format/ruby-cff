    CFF::Error

    class CFF::ValidationError

    ValidationError is raised when a CFF file fails validation. It contains details of each failure that was detected by the underlying JsonSchema library, which is used to perform the validation.

    Additionally, the invalid_filename flag is used to indicate whether the CFF file is named correctly. This is only used when validating a File; validating a Index directly will not set this flag to true.

    Attributes

    errors[R]

    The list of JsonSchema::ValidationErrors found by the validator.

    invalid_filename[R]

    If a File was validated, was its filename invalid?
